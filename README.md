# node-fetch-example

Makes a client side fetch request via an [Express](https://expressjs.com/) route to a third party API using [node-fetch](https://www.npmjs.com/package/node-fetch).

## Installation

```sh
$ npm install
```

## Express

```sh
$ node index.js
```

or

```sh
$ npm run start
```

Starts a server at `http://localhost:3000`
